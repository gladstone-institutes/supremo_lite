# supremo_lite

A lightweight memory first, model agnostic version of [SuPreMo](https://github.com/ketringjoni/SuPreMo).

## Installation

```bash
$ git clone https://github.com/gladstone-institutes/supremo_lite.git
$ cd supremo_lite
$ pip install .
```
