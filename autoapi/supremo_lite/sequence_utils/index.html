supremo_lite.sequence_utils

Sequence transformation utilities for supremo_lite.

This module provides functions for encoding, decoding, and manipulating DNA sequences.

supremo_lite.sequence_utils.encode_seq(seq, encoder=None)[source]

Convert a nucleotide string to a one-hot encoded tensor/array.

Parameters:

  • seq – A string of nucleotides or a list of such strings

  • encoder – Optional custom encoding function. If provided, should accept a single sequence string and return encoded array with shape (4, L).

Returns:

A tensor/array with shape (4, L) for a single sequence or (N, 4, L) for a list, where L is the sequence length and N is the number of sequences.

Encoding scheme (default):

‘A’ = [1, 0, 0, 0] (first channel) ‘C’ = [0, 1, 0, 0] (second channel) ‘G’ = [0, 0, 1, 0] (third channel) ‘T’ = [0, 0, 0, 1] (fourth channel) ‘N’ = [0, 0, 0, 0] (all channels zero)
